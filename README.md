# Web Radio Auto - App Android

App wrapper per la webapp Web Radio Auto.
Risolve il problema dei comandi al volante su autoradio Android.

## Setup

1. Modifica l'URL in `app/src/main/java/com/webradio/auto/MainActivity.java`
2. Push su GitHub → l'APK viene compilato automaticamente
3. Scarica l'APK dalla tab Actions → Artifacts
4. Installa sull'autoradio
